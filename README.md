# Instructions

## 1. Build Frontend

1. `cd frontend`
2. `npm install` or `yarn install`
3. `npm run build` or `yarn build`

## 2. Run Flask Server

1. `cd backend`
2. `pip install -r requirements.txt`
3. `flask run` for development server or `gunicorn -w 4 -b {host}:{port} app:app` for production server
